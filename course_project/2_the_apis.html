Requesting information from Musixmatch follows a standard process for making 'GET' requests.


 function process(data) {
   console.log(data);
 }

 var request = document.createElement('script');

 var url = 'https://api.musixmatch.com/ws/1.1track.search?format=jsonp&callback=process&q_artist=iron%20and%20wine&quorum_factor=1&apikey=d91e6ac977972cb40004b439e314009c';

 request.setAttribute('src', url);

 document.querySelector('body').appendChild(request);

The Soundcloud API provides a software development kit (SDK) that includes convenient methods for not only making requests but also for streaming sounds. The SDK also utilizes ES6 features that allow you to chain a callback and error handling onto a request.
