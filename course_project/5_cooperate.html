Ideally, sounds from SoundCloud would be able to be paired with cooresponding lyrics from Musixmatch. However, both databases are far from comprehensive. Further, data can be incorrectly labeled such that tracks cannot be predictably located. Though the player will attempt to locate both a sound and set of lyrics related to a search term, it would have to accommodate for situations in which one or both requests would not return data.

Below, you can enter an artist and/or track name to search both databases. If a response from either source indicates a possible match, its icon will become bold. Requests that may include both a sound and lyrics would be visualized by both icons becoming bold. After a few seconds, the form resets for repeated queries. You can open the browser console to inspect responses.
