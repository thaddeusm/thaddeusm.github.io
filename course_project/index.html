Combining APIs

Why? How?

Opportunities abound for front-end web developers to improve their skills through small projects. Client-side games, data visualizations, and browser-based tools allow you to learn by doing, applying existing knowledge practically and stretching skills when encountering errors. Whether developing a portfolio of these types of projects helps a new developer find employment is unclear, but the process is certainly beneficial to learning. Code schools and 'bootcamps' develop curricula that are based around projects, understanding that putting new information into practice is essential in supporting career readiness.

This website aims to provide a walkthrough of an example project that draws upon two databases related to music: Soundcloud and Musixmatch. The project will explore opportunities for and limitations to integration between a database of sounds and a database of lyrics.

Thaddeus McCleary - 2017
