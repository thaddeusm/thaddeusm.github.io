My Officials

Built a web application within an Intermediate Development Course at the University of Missouri that employs AJAX in JavaScript to send user-provided information to the Google Civic Information API: and returns information about the elected representatives at the federal and state levels for a given address.

Includes native CSS dropdowns with pseudo classes and JavaScript event listeners for touch support. Coded a native JavaScript lightbox effect for images.

Making Connections

Through a six-part online tutorial with blog-style posts, code samples, and demonstration videos, built a web player that queries Soundcloud and Musixmatch databases to attempt to display lyrics while listening to a song. Developed UI from scratch with mobile responsiveness. Completed as a final project for a Interactive Web Development with JavaScript course at the University of Missouri.
